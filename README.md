## Installation


1. Clone the repo

```sh
https://github.com/acelyavul/space-tourism-website.git
```

2. Install all the dependencies inside the project folder

```sh
npm i
```

3. Start Development

```sh
npm run watch:sass
```

4. Build

```sh
npm run build:css
```
